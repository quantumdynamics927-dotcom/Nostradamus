# Period Sources Index

## Source Hierarchy

| Tier | Type | Trust | Usage |
|------|------|-------|-------|
| **1** | 1555/1568/1605 editions | **Primary** | Canonical text |

## Period Chronicles (Historical Matching)

- **de Thou** - *Historia Sui Temporis* (1609) - French Wars of Religion
- **L'Estoile** - *Mémoires-Journaux* (1574-1611) - Daily Parisian life
- **Froissart** - *Chroniques* (c.1480) - Earlier period
- **Belleforest** - *Annales de France* (1559-1572) - Chronicles

## Source Policy

1. **Start from earliest French** - Every quatrain keyed to 1568/1605 page+line
2. **Document every transformation** - Normalization logged, reversible
3. **Modern sources = commentary** - Store separately, never mix into core matching
4. **Validation reproducible on Tier 1** - Show French → modules → mapping
